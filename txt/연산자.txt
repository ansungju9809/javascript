연산자란?
어떠한 작업을 컴퓨터에 지시하기 위한 기호

대입 연산자 기호:=
어떠한 변수안에 값을 대입하는 연산자이다.

비교 연산자
주어진 값을 같은지, 다른지, 큰지, 작은지를 구분하는 연산자다.
비교연산자의 결과는 불린(boolean)으로 표현한다.
출력은 맞으면 true 틀리면 false로 출력한다.

동등 연산자  기호: ==
좌항과 우항을 비교해 같으면 true 다르면 false가되는 연산자.
예: alert(1==2)--> false
    alert(1==1)--> true
숫자뿐만 아니라 문자도 가능

일치 연산자  기호: ===
좌항과 우항이 정확하게 같아야지만 true 조금이라도 다르면 false 가 된다.
예: alert(1=="1") --> true
    alert(1==="1") --> false

undefined 와 null의 차이점
undefined는 값이 존재 하지않다고 프로그램이 정하는거고
null은 프로그래머가 값이 없는 것을 의도한 것이다.

!=이란?
!= 은 부정이란 표현이다.
예: alert(1!=1) --> false
    alert(1!=2) --> true
>,<,>=,<= 란?
부등호이다.
>는 좌항이 우항보다 크냐 이다. <는 반대
>= 는 좌항이 우항보다 크거나 같냐 이다. <=는 반대
